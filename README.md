# FunGroww
🚀 FunGrow – Freelancing Platform

FunGrow is a full-stack freelancing web application that connects clients with freelancers specializing in technical services such as website development, front-end development, and back-end development.

It provides secure authentication, project management, search & filter functionality, and smooth user experience.
